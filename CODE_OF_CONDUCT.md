# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying precise about security and correctness risks.
- Prefer actionable feedback with reproduction steps, tests, or documentation links.

## Unacceptable Behavior

Harassment, discriminatory language, threats, doxxing, sustained disruption, or public disclosure of sensitive vulnerability details before maintainers have had a reasonable chance to respond are not acceptable.

## Enforcement

Maintainers may edit or remove comments, close issues, or restrict participation when behavior violates this code. Report concerns through the same private channels listed in `SECURITY.md`.
